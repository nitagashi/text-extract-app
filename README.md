# OCR Image Text Extractor

A modern React application that extracts text from images using OCR (Optical Character Recognition) powered by Tesseract.js.

---

## Features

- Extract text from uploaded images
- Modern glassmorphism UI
- Live OCR processing progress
- Image preview before extraction
- Responsive design
- Built with React

---

## Tech Stack

- React
- Tesseract.js
- CSS3

---

## Installation

Clone the repository:

```bash
git clone <your-repository-url>
```

Navigate into the project folder:

```bash
cd <project-folder>
```

Install dependencies:

```bash
npm install
```

---

## Run the Project

Start the development server:

```bash
npm start
```

The app will run on:

```bash
http://localhost:3000
```

---

## Usage

1. Upload an image containing text
2. Click **Extract Text**
3. Wait for OCR processing to finish
4. View the extracted text output

---

## Supported Image Types

- PNG
